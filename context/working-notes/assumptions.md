# Working Assumptions

- The GitHub repository `VehicleOS` exists remotely and is now scaffolded locally.
- The architecture narrative should support product demos, technical storytelling, and contributor review.
- The first deliverable should optimize for a strong architecture narrative and a credible MVP.
- Postgres is the default system of record unless a later ADR disproves that choice.
- Human approval remains mandatory for any action that affects money, appointments, compliance, or external submissions.
