## Choose your path

| | **Owners** | **Builders** |
|---|------------|--------------|
| **On site** | **Use the app** | **Run it yourself** |
| **Tagline** | Let Vehicle OS work in the background. | Run Vehicle OS on your machine — full control, open core. |
| **Surfaces** | Web, mobile, Vehicle OS Connect (desktop) | Self-host, CLI, integrations |
| **Price (now)** | Free · early access | Free · MIT |
| **Price (later)** | Subscription tiers when product matures | Still free |

**Owners** — managed product; no code to run.  
**Builders** — clone, self-host, extend.
