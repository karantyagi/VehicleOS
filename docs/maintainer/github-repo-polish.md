## Social preview (LinkedIn / X / Slack)

GitHub does not expose social preview upload via API. Set manually:

1. Repo → **Settings** → **General**
2. **Social preview** → **Upload an image…**
3. Use **1280×640** PNG/JPG (e.g. screenshot of [vehicleos.app](https://vehicleos.app) hero, or a branded card with logo + tagline)
4. Save

Test: paste `https://github.com/karantyagi/VehicleOS` into LinkedIn post composer and confirm the card looks right.

**Tip:** Match marketing site copy — *Operational memory for long-lived ownership* / *Explainable AI maintenance for vehicle ownership*.
